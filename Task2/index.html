Task Description

Objective: Set up Terraform and launch an AWS EC2 instance.

Steps

  1. Downloaded and extracted Terraform.
  2. Added Terraform to the system PATH.
  3. Verified installation via terraform --version.
  4. Created main.tf with AWS provider and EC2 resource.
  5. Ran terraform init, plan, and apply.
  6. Destroyed resources with terraform destroy.
